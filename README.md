# Banco Quântico Project

## Funcionalidade
Este lindo app bancario permite aos usuários:
- Transferir dinheiro entre os usuários
- Solicitar empréstimo 
- Apagar a conta 
- balanço total doque entra e sai da conta

## Habilidades
neste projeto trabalhamos com:
- Matrizes 
- Números 
- Datas 
- Cronômetros 
- Fake "login"
